* Example project

Here's how you would run this HTTP echo JSON pretty printer service:

#+begin_src sh :results verbatim
mvn function:run > /tmp/output.log &
echo $!
#+end_src

#+begin_src sh :results verbatim :exports results
cat /tmp/output.log | tail -n 4
#+end_src
