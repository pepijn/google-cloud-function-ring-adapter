* Example project

Here's how you would run this HTTP echo JSON pretty printer service:

#+name: start-server
#+begin_src sh :eval no
mvn function:run
#+end_src

#+begin_src sh :results verbatim :noweb yes :exports none
<<start-server>> > /tmp/output.log &
echo $!
sleep 4
#+end_src

#+name: pid
#+RESULTS:
: 50831
